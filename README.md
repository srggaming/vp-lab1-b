# Web Programming Lab 1

Spring Boot application for managing chefs and dishes.

## Requirements

- Java 17+
- Maven 3.6+

## Running the Application

```bash
mvn spring-boot:run
```

Access the application at: `http://localhost:8080/listChefs`

## Technologies

- Spring Boot 3.1.5
- Thymeleaf
- Maven
- Lombok
